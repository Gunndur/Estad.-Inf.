Contexto:

En una planta qumica hay dos mquinas que envasan detergentes industriales en bidones con un volumen de producto que sigue una distribucin normal con desviacin estndar de 1 litro. La ingeniera a cargo de la planta debe asegurar que los bidones se estn llenando con una media de 10 litros. Pero ella tiene la sospecha de que hay desviaciones en esta media, lo que piensa confirmar usando una muestra aleatoria de 100 envases (50 de cada una de las mquinas). Tambin cree que hay diferencia en el cumplimiento del volumen requerido entre la mquina ms antigua y la ms moderna, que han de andar por el 90% y 96% de los bidones, respectivamente.


Pregunta 1:

Si la ingeniera piensa rechazar la hiptesis nula cuando la muestra presente una media menor a 9,78 litros o mayor a 10,22 litros, cul es la probabilidad de que cometa un error de tipo I? Para responder, generen un grfico de la distribucin muestral de las medias hipotetizada en donde se marquen las zonas correspondientes a la probabilidad solicitada, para luego, basndose en este grfico, calcular el rea correspondiente. Tome como ejemplo los scripts presentados en la lectura sobre poder estadstico.

Un error de tipo I ocurre cuando se rechaza la hiptesis nula cuando esta es verdadera, la ingeniera piensa rechazar la hiptesis nula cuando la muestra este fuera del intervalo de confianza [9.78,10.22], es decir, cuando la media muestral caiga fuera de ese rango se cometera el error tipo I, para calcular la probabilidad de esto, hay que calcular el rea que se encuentra fuera del intervalo de confianza en la curva normal. 
mu_0 <- 10  # H0
sigma <- 1  # Desvest
n <- 50  # Muestra
errorEstndar <- sigma / sqrt(n)  # Error estndar de la media muestral

# Valores crticos para el rechazo de la hiptesis nula
limiteInf <- 9.78
limiteSup <- 10.22

# Eje x
x <- seq(9, 11, length.out = 1000)

# Distribucin normal bajo la suposicin de que H0 es verdadera
y <- dnorm(x, mean = mu_0, sd = errorEstndar)

# Clculo de las reas correspondientes a error tipo I
errorAreaIzq <- pnorm(limiteInf, mean = mu_0, sd = errorEstndar)
errorAreaDer <- 1 - pnorm(limiteSup, mean = mu_0, sd = errorEstndar)
errorTipoI <- errorAreaIzq + errorAreaDer

df <- data.frame(x = x, y = y)
ggplot(df, aes(x, y)) +
  geom_line(color = "blue", linewidth = 1) +
  # reas que representan el error tipo I
  geom_area(data = subset(df, x <= limiteInf), aes(x, y), fill = "red", alpha = 0.5) +
  geom_area(data = subset(df, x >= limiteSup), aes(x, y), fill = "red", alpha = 0.5) +
  # Lneas que representan los lmites
  geom_vline(xintercept = limiteInf, linetype = "dashed", color = "red") +
  geom_vline(xintercept = limiteSup, linetype = "dashed", color = "red") +
  ggtitle("Distribucin Muestral bajo H0 y Error Tipo I") +
  xlab("Media muestral") + ylab("Densidad de probabilidad") +
  theme_minimal()

# Mostrar la probabilidad de error tipo I
cat(sprintf("Probabilidad de error tipo I: %.4f", errorTipoI, errorTipoI * 100))
## Warning in sprintf("Probabilidad de error tipo I: %.4f", errorTipoI, errorTipoI
## * : one argument not used by format 'Probabilidad de error tipo I: %.4f'
## Probabilidad de error tipo I: 0.1198

Respuesta:

La probabilidad de cometer un error tipo I es del 0.1198, en otras palabras, hay aproximadamente un 12% de rechazar la hiptesis nula cuando esta es verdadera.

Pregunta 2:

Si el verdadero volumen medio de los bidones fuera de 10,2 litros, cul sera la probabilidad de que la ingeniera, que obviamente no conoce este dato, cometa un error de tipo II? Un error de tipo II ocurre cuando se rechaza la hiptesis alternativa cuando esta es verdadera, en otras palabras, se acepta la hiptesis nula cuando esta es falsa, para obtener la probabilidad, hay que calcular el rea debajo de la curva normal que se genera tomando 10.2 como la media verdadera (o hiptesis alternativa) y dentro del intervalo de confianza, ya que al tomar la suposicin de que la hiptesis alternativa es verdadera, la hiptesis nula es falsa, y el rea dentro del intervalo de confianza es aceptar esa hiptesis nula. 
mu_01 <- 10.2  # Hiptesis nula (media verdadera es de 10.2)

# Distribucin normal asumiendo que la hiptesis nula es verdadera
y1 <- dnorm(x, mean = mu_01, sd = errorEstndar)

# Clculo del rea de error tipo II (probabilidad de no rechazar H0 cuando H1 es verdadera)
errorTipoII <- pnorm(limiteSup, mean = mu_01, sd = errorEstndar) - pnorm(limiteInf, mean = mu_01, sd = errorEstndar)


df1 <- data.frame(x = x, y = y1)

# Graficar las distribuciones

  # Distribucin bajo mu_0 (media = 10)

  # Lneas que representan los lmites

  geom_vline(xintercept = limiteSup, linetype = "dashed", color = "red") +

Pregunta 3:

Como no se conoce el verdadero volumen medio, genere un grfico del poder estadstico con las condiciones anteriores, pero suponiendo que el verdadero volumen medio podra variar de 9,5 a 10,5 litros. Hay un ejemplo de este tipo de grfico en la lectura sobre poder estadstico.

Para la creacin del grfico, se utiliz como base solamente, un grfico que se encuentra de ejemplo en la lectura 4 del ramo. 
# Datos sobre el Tamao de efecto
n <- 50
alfa <- 0.05
media_nula <- 10
sigma <- 1

# Clculo del poder
medias_verdaderas <- seq (9.5 , 10.5 , 0.01)
deltas <- medias_verdaderas - media_nula
deltas <- round(deltas, 2)
deltas_normalizados <- deltas / sigma

f_pzb <- function ( x ) pwr.norm.test (x , n = n , sig.level = alfa ,alternative = "two.sided") [["power"]]
poder_bilat <- sapply ( deltas_normalizados , f_pzb )
df<-data.frame(deltas_normalizados,poder_bilat)
#(df, x="deltas_normalizados" , y="poder_bilat", group = 1, color= "black")

plot(df$deltas_normalizados,df$poder_bilat,type = "b", main = "Poder v/s tamao",
     ylab = "Poder", xlab = "Tamao de efecto")

Respuesta:

Desde el grafico anterior se puede concluir que a mayor tamao de efecto, mayor es el poder estadistico presente.

Pregunta 5:

Alcanzara esta muestra para detectar la diferencia que la ingeniera sospecha que existe entre las dos mquinas de la planta con al menos las mismas probabilidades de cometer errores?

Debido a que ahora se busca encontrar el tamao de la muestra necesario para detectar la diferencia entre ambas maquinas, se debe utilizar un test para 2 probabilidades con muestras del mismo tamao. Por lo tanto se usara la funcion pwr.2p.test() con las siguientes variables:

h = ES.h(0.9,0.96) (tamao de efecto medido con h de Cohen en base a las proporciones).

n = NULL (Tamao de muestra buscado).

sig.level = 0.05.

power = 0.75.

alternative = two.sided debido a que se busca detectar si existe una desviacion en el cumplimiento del volumen requerido de ambas maquinas.

# POTENCIA DE LA PRUEBA DE LA DIFERENCIA DE DOS PROPORCIONES
size<-pwr.2p.test(h=ES.h(0.90,0.96), n=NULL , sig.level=0.05 , power=0.75, alternative = "two.sided")
print(size)
